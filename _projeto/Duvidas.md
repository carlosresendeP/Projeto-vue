# Duvidas do desafio

## 1. Banco de Dados

Vamos utilizar o Postgres. Somente essa informação? Docker ou Nuvem?
Decisão: Optei por utilizar o Postgres na nuvem com o Supabase para agilidade e facilidade de acesso.

## 2. Framework CSS

Vue 3 com Quasar. Poderia usar Tailwind?
Optei por utilizar o padrão do Quasar (que possui seu próprio sistema de grid e utilitários) para seguir estritamente o solicitado

## 3. Responsavel

No desafio é obrigatório o cadastro de clientes e demandas, mas não é obrigatório o cadastro de usuários, No caso o responsavel, por exemplo dentro do setor Frontend, 'Maria', no setor de banco de dados 'João' e assim por diante, correto?
Decisão: Optei por não cadastrar usuários, pois não é obrigatório no desafio. Mantive somente o campo responsavel como string. podendo ser qualquer nome.
